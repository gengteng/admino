# Admino

[WIP] 个人学习项目，实现一个后台管理系统。

## RoadMap

* [ ] 单页应用（SPA），实现前后端分离
* [ ] 后端框架使用 actix-web 2.0+
* [ ] 前端使用基于 Angular 的 ngx-admin/nebular（移动端也能很好的支持）
* [ ] 数据库使用 PostgreSQL + Redis
* [ ] 定义良好层次清晰的错误处理
* [ ] 参考 RBAC3 概念实现完整的权限管理模块
